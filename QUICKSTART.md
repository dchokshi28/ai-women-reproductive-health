# Quick Start

## Setup
1. Install PostgreSQL and create database: `womens_health_db`
2. Run: `setup_new.bat`
3. Edit `new_backend\.env` with your database password
4. Run: `start_new.bat`

## Access
- Frontend: http://localhost:3000
- Backend: http://localhost:8000
- API Docs: http://localhost:8000/docs
